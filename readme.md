# Fin-Slang
Transforms common internet slangs into a more formal words.

```
gr8 -> great
plz -> please
imo -> in my opinion
..etc
```


## Installation

```
npm i --save fin-slang
```

## Usage

It'll work just by importing it.

```javascript
import * as Fin from "finnlp";
import "fin-slangs";
// now when you instantiate a new input, slangs will be transformed
```
